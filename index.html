The purpose of the project is to help Rosa find an ideal bacterial species to make synthetic beef. She believes in can be found in some people's belly buttons. This dashboard allows participants to select their id number and see which bacterial species are in their belly buttons. Researchers are able to access the data as well. The charts are interactivea and change to match each id number selected.
